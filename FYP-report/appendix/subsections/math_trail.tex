\chapter{Mathematical Audit Trail}
\label{appendix:math_trail}

This appendix serves as a complete visual breakdown of what a typical MERIT~\cite{merit_repo} metric calculation looks like
for a given candidate. In this case, we consider a programming language metric, Python, this is an extensible
metric derived from a job description (not a hardcoded baseline metric). Here, we show the recruiter the
complete logic behind how the strength of each data source is calculated, and the weights that are applied to 
each source when later performing Bayesian Evidence Fusion. This trail also accounts for heuristic decay,
batch normalisation and also the display of Shapley Values to inform the recruiter whether our score 
was driven by demonstrated evidence, or claimed evidence.

\begin{figure}[htpb]
    \centering
    \includegraphics[width=0.85\textwidth]{implementation/screenshots/math_trail_phase_0_1.png}
    \caption[Math Audit Trail: Phases 0 \& 1 (Data Source Strength \& Skill Decay)]{Phase 0 and Phase 1 of the Math Audit Trail 
    for the Python metric (extensible metric derived from a JD). The calculations for the strength of each data source are shown,
    and skill decay is applied specifically to the GitHub source. Note that we do not perform this operation for self-reported data
    to account for potential stretching of the timeline for projects.}
    \label{fig:audit_trail_phase_1}
\end{figure}

\begin{figure}[htpb]
    \centering
    \includegraphics[width=0.85\textwidth]{implementation/screenshots/math_trail_phase_2.png}
    \caption[Math Audit Trail: Phase 2 (Bayesian fusion \& Shapley Values)]{Phade 2 of the Math Audit Trail for the 
    Python metric. This phase dispplays the Bayesian Evidence Fusion process as well as shapley value calculations.
    For bayesian fusion, we use the strength of the signal $Str$ and the trust value of the signal $Tr$ to update
    the alpha and beta parameters of the Beta distribution. We compute Shapley Values for each data source to determine
    the contribution of each source to the final score.}
    \label{fig:audit_trail_phase_2}
\end{figure}
