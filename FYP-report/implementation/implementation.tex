\chapter{The Architecture \& Implementation of MERIT}
\label{ch:implementation}

In this chapter, we cover both the architectural design, as well as the implementation of MERIT.
We start with a high-level system overview, presenting the three key stages of the pipeline (Ingestion, Scoring, Explainability).
From there, we will cover each of these stages in more detail, going over the notable contributions to each stage that
differ from other existing screening systems. From there, we will conclude the chapter with bias and algorithmic safeguards
that prevent candidates from gaming the system, as well as the anti-bias measures that we take to ensure a fair assessment of 
candidate potential, devoid of personal bias. We will now cover the specific design choices and mathematical models that MERIT
leverages across these three phases to ensure extensibility, multi-source fusion and explainability.
