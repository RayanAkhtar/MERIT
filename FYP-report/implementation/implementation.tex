\chapter{Implementation \& Architecture}

This chapter outlines the implementation and architecture of MERIT. This involves
the transition from theoretical frameworks establishes in chapter 2, to a functional,
full-stack implementation of a multi-source, extensible and green-box candidate screening
system tailored to technical recruitment.
For our implementation, we built a decoupled microservice pattern, leveraging Next.js for
the frontend due to its ease of use and speed of development. For similar reasons, we 
also opted for a Flask backend, noting its key advantage of large data processing capabilities 
and its rich ecosystem of NLP libraries. 
The implementation involves three key layers: an extraction layer, a configuration layer,
and an execution layer. We will cover specific technologies, design choices and the mathematical
models that MERIT leverages.
